## [숫자카드나누기](https://school.programmers.co.kr/learn/courses/30/lessons/135807)
int[] 철수카드, int[] 영희카드가 주어졌을 때 철수카드의 공약수가 영희카드의 숫자를 하나도 나눌 수 없거나, 영희카드의 공약수가 철수카드를 하나도 나눌 수 없는 숫자 a를 구하라 

### 1. 이해
- 숫자 a는 무조건 1보다 크다 
- 숫자 a는 철수 혹은 영희 카드를 모두 나눌 수 있고, 영희의 모든 카드를 나눴다면 철수의 모든 카드가 나눠지면 안된다. 그리고 반대 조건도 해당한다. 
- 두 조건 모두 충족하는 숫자가 없는 경우 0을 반환한다

### 2. 계획
- 철수 그리고 영희 카드를 모두 PriorityQueue로 집어넣는다. 
- 철수카드의 모든 카드를 나눌 수 있는 공약수를 찾는다. 
- 철수카드의 공약수 중 영희카드를 하나도 나눌 수 없는 숫자를 찾는다 
- 영희카드의 모든 카드를 나눌 수 있는 공약수를 찾는다. 
- 영희카드의 공약수 중 철수카드를 하나도 나눌 수 없는 숫자를 찾는다
- 공약수를 찾는 방식은 첫 번째 숫자와 두 번째 숫자의 공약수들을 찾고
- 해당 공약수로 나머지 숫자를 나눠보는 방식 
- 공약수 후보배열을 구하는 공식은 n = 2; n <= Math.sqrt(number); n += 1 까지 반복하고 number % n == 0 이라면 n과 number / n을 push 마지막으로 number를 push 
- 공약수 후보배열로 나머지 카드를 순회하며 % n != 0 이라면 삭제한다 
- 나온 공약수 배열로 상대방 카드를 순회하고 % n == 0 이라면 삭제한다. 
- 두 상대의 가장 큰 값을 비교해 큰 값을 반환하고 
- 공약수 배열이 모두 size == 0 이라면 0을 반환한다
