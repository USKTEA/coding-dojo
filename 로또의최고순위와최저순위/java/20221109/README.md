6개 숫자로 이루어진 로또가 있다. 몇 개의 숫자가는 오염이 되어 알아볼수 없으나 최고 순위와 최저 순위를 알아보고 싶다. 
당첨번호가 주어졌을 때 오염된 로또 번호를 이용해 최고 순위와 최저 순위를 알아보자 
### 1. 이해
- 길이가 6인 정수 배열 2개가 주어진다. 하나는 오염된 로또번호 하나는 당첨번호. 
- 오염된 숫자는 0으로 처리된다.
- 숫자 0은 당첨 번호가 될수도 아닐수도 있다. 
- 숫자 0이 당첨 번호일때와 당첨 번호가 아닐 때의 순위를 정수 배열로 반환해라. 
- 숫자 6개 일치 1등, 5개 2등, 4개 3등, 3개 4등, 2개 5등 그 외 낙첨
### 2. 계획
- Arrays.reduce를 돌아서 더한다 
- 번호가 0인 경우는 냅둔다. 
- 0이 아닌 숫자와 win_nums와 비교해 최저 순위를 확보
- 0의 갯수 만큼 점수를 더해 최고 순위를 확보한다.
